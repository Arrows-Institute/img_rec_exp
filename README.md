中学生・高校生の初めてのプログラミング「画像認識プログラム」です。
コードはPythonによって書かれており、ただ実行するだけで「画像認識」を体験することが出来ます。自分の画像で認識を行いたいという場合も、HSVを利用して一定の領域の色を認識することが出来ます。
